# Release v1.2.0 - Galactic Navigation Update

Twin Marks AI がさらに進化しました！今回のアップデートでは、3Dナレッジグラフの探索体験を大幅に強化し、SF映画のような没入感のあるインターフェースを追加しました。

## 🚀 新機能

### 🌌 3Dグラフ ナビゲーション (Galactic Navigation)
*   **検索結果ツアー**: 関連するページ（星）を順番に巡回できるナビゲーションボタン（`<` `>`）を追加しました。広大な知識の宇宙で迷うことなく、重要な情報へたどり着けます。
*   **自動カメラ制御**: 次の結果へ移動する際、カメラが滑らかに対象の星へズームインします。

### 🛸 SF風 UIデザイン (Sci-Fi Interface)
*   **ハイライトエフェクト**: 検索でヒットした星は**シアン（水色）**の強力なオーラを纏い、一目で識別できるようになりました。
*   **ホバー拡大**: 星にマウスを乗せると、スプライトがふわっと拡大し、選択感を高めます。
*   **高機能ツールチップ**: マウスオーバー時の情報表示を刷新。半透明のガラス風パネルにスキャンラインが走る、未来的なデザインを採用しました。

### ⚙️ その他の改善
*   **デフォルト設定の調整**: 
    *   3Dグラフのリンク密度（閾値）を調整し、より多くのつながりが見えるようにしました。
    *   スクリーンショット機能（画像解析）のデフォルトを**OFF**に変更し、トークン消費を抑える設定にしました（設定からONにできます）。

## 📦 インストール方法

1.  [Releasesページ](https://github.com/dma-cmyk/twin-marks-ai/releases)から `twin-marks-ai-v1.2.0.zip` をダウンロード。
2.  解凍して `dist` フォルダを取り出す。
3.  Chromeで `chrome://extensions/` を開き、「デベロッパーモード」をON。
4.  「パッケージ化されていない拡張機能を読み込む」から `dist` フォルダを選択。

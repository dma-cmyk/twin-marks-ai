# Release v1.3.0 - AI Knowledge Graph & RAG Update

Twin Marks AI が大幅に進化しました！
今回のアップデートでは、ブックマークを知識源として活用する **RAG (AI Chat)**、3D空間での探索を強化した **ナレッジグラフ**、そして散らかったタグを整理する **AIタグ最適化** 機能が追加されました。

## 🚀 新機能 (New Features)

### 🧠 RAG (Retrieval-Augmented Generation)
*   **AIチャット機能**: 保存されたブックマークの内容を知識源として、AIがあなたの質問に回答します。「あの記事どこだっけ？」だけでなく、「Reactのパフォーマンス最適化について要約して」といった質問にも、あなたのライブラリから答えを導き出します。
*   **文脈理解**: 単なるキーワード検索を超え、意味に基づいた回答生成を行います。

### 🌌 ナレッジグラフ & クラスタリング (Knowledge Graph)
*   **AI自動クラスタリング**: `ml-kmeans` を導入し、意味的に近いページを自動でグループ化。AIが各クラスタの内容を分析し、適切なラベル（カテゴリ名）を自動付与します。
*   **2D / 3D ビュー切り替え**: 用途に合わせて、没入感のある **3D Galaxy View** と、全体像を把握しやすい **2D Network View** を使い分けられるようになりました。
*   **SF風 UIデザイン**: 検索結果がシアン色に発光し、マウスオーバーで詳細が表示されるSFモニタ風インターフェースを採用しました。

### 🏷️ タグ最適化 (Tag Optimization)
*   **AIタグ整理**: 表記揺れ（例: "Javascript" vs "JS"）や、重複した意味を持つタグをAIが検出し、統廃合を提案します。
*   **一括適用**: 提案された変更を確認し、ワンクリックでライブラリ全体のタグを一貫性のある状態に整理できます。

### 🛸 ギャラクティック・ナビゲーション (Galactic Navigation)
*   **検索結果ツアー**: 3D空間内で検索結果のページ（星）を順番に巡回できるナビゲーションボタン（`<` `>`）を追加。カメラが自動で対象へズームインします。

## 🛠️ 技術的な改善 (Improvements)
*   **UIコンポーネント刷新**: `lucide-react` (アイコン), `react-resizable-panels` (パネルリサイズ) を採用し、モダンで操作性の高いUIに刷新しました。
*   **パフォーマンス**: グラフ描画エンジンとデータ処理の最適化を行いました。

## 📦 インストール方法

1.  Assets から `twin-marks-ai-v1.3.0.zip` をダウンロード。
2.  解凍して `dist` フォルダを取り出す。
3.  Chromeで `chrome://extensions/` を開き、「デベロッパーモード」をON。
4.  「パッケージ化されていない拡張機能を読み込む」から `dist` フォルダを選択。

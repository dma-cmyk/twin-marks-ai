# Release v1.3.1 - AI Notebook, Smart Categorization & Gemini 2.5

Gemini 2.5 Flash-Lite の導入と分析フローの効率化に加え、学習効率を高める「AIノートブック」機能と、ライブラリ全体を整理する「スマートカテゴリ同期」を追加しました。

## 🚀 新機能 (New Features)

### 📝 AIノートブック (Smart Notebook Panel)
*   **統合テキストエディタ**: 保存済みページごとに専用のメモ帳を実装しました。記事を読みながら要点をまとめたり、コードスニペットを保存するのに最適です。
*   **集中執筆モード**: フルスクリーン表示、行番号表示、文字数カウント、折り返し設定など、本格的なエディタ機能を搭載。
*   **Markdownフレンドリー**: シンプルなテキスト形式で保存されるため、Markdown記法での記述にも対応しています。

### 🏷️ スマートカテゴリ同期 & 一括操作
*   **AI自動カテゴリ再編**: ワンクリックでライブラリ内の全ブックマークを約20個の主要カテゴリ（例: 「プログラミング」「デザイン」「ニュース」など）に自動分類・同期します。これにより、RAG（AIチャット）の検索精度と文脈理解が向上します。
*   **一括編集機能**: 複数のアイテムを選択し、タグの一括追加・削除や、アイテムの一括削除が可能になりました。
*   **高度なフィルタリング**: タグやAIカテゴリによる絞り込み（OR検索）に対応し、目的の情報を素早く見つけられます。

### 🎨 3Dグラフ・テーマ (Visual Themes)
*   **3つの没入テーマ**: ナレッジグラフの3Dビューに「Universe」「Cyberpunk」「Deep Sea」のテーマを追加しました。気分や用途に合わせて切り替え可能です。

### 🔍 検索・ソート機能の強化
*   **意味的ソート (Semantic Sort)**: 検索クエリとの「意味的な近さ」で検索結果を並び替える機能を追加しました。キーワードが一致しなくても、文脈が近いページが上位に表示されます。

### 📸 スクリーンショット優先の分析フロー
*   **高効率・高精度**: ページの全文テキストの代わりに、スクリーンショット画像とメタデータ（Title, H1, Description）を優先的に使用して要約・タグを生成するようになりました。
*   **コスト削減**: 送信トークンを最小限に抑えつつ、Gemini 2.5 の強力な視覚能力により、記事の本質をより的確に捉えられます。
*   **ハイブリッド構成**: 画像が使えない場合は自動的に従来通りの全文テキスト分析にフォールバックします。

### 🤖 最新モデルへの対応
*   **Gemini 2.5 Flash-Lite**: 推奨生成モデルを最新の `gemini-2.5-flash-lite` に更新しました。
*   **Embedding 001**: 推奨エンベディングモデルを `text-embedding-001` に更新しました。

## 🛠️ バグ修正 & その他
*   UI上のバージョン表記不一致を修正。
*   README の情報を最新の推奨設定に合わせて更新。

# Release v1.3.0 - AI Knowledge Graph & RAG Update

Twin Marks AI が大幅に進化しました！
今回のアップデートでは、ブックマークを知識源として活用する **RAG (AI Chat)**、3D空間での探索を強化した **ナレッジグラフ**、そして散らかったタグを整理する **AIタグ最適化** 機能が追加されました。

## 🚀 新機能 (New Features)

### 🧠 RAG (Retrieval-Augmented Generation)
*   **AIチャット機能**: 保存されたブックマークの内容を知識源として、AIがあなたの質問に回答します。「あの記事どこだっけ？」だけでなく、「Reactのパフォーマンス最適化について要約して」といった質問にも、あなたのライブラリから答えを導き出します。
*   **文脈理解**: 単なるキーワード検索を超え、意味に基づいた回答生成を行います。

### 🌌 ナレッジグラフ & クラスタリング (Knowledge Graph)
*   **AI自動クラスタリング**: `ml-kmeans` を導入し、意味的に近いページを自動でグループ化。AIが各クラスタの内容を分析し、適切なラベル（カテゴリ名）を自動付与します。
*   **2D / 3D ビュー切り替え**: 用途に合わせて、没入感のある **3D Galaxy View** と、全体像を把握しやすい **2D Network View** を使い分けられるようになりました。
*   **SF風 UIデザイン**: 検索結果がシアン色に発光し、マウスオーバーで詳細が表示されるSFモニタ風インターフェースを採用しました。

### 🏷️ タグ最適化 (Tag Optimization)
*   **AIタグ整理**: 表記揺れ（例: "Javascript" vs "JS"）や、重複した意味を持つタグをAIが検出し、統廃合を提案します。
*   **一括適用**: 提案された変更を確認し、ワンクリックでライブラリ全体のタグを一貫性のある状態に整理できます。

### 🛸 ギャラクティック・ナビゲーション (Galactic Navigation)
*   **検索結果ツアー**: 3D空間内で検索結果のページ（星）を順番に巡回できるナビゲーションボタン（`<` `>`）を追加。カメラが自動で対象へズームインします。

## 🛠️ 技術的な改善 (Improvements)
*   **UIコンポーネント刷新**: `lucide-react` (アイコン), `react-resizable-panels` (パネルリサイズ) を採用し、モダンで操作性の高いUIに刷新しました。
*   **パフォーマンス**: グラフ描画エンジンとデータ処理の最適化を行いました。

## 📦 インストール方法

1.  Assets から `twin-marks-ai-v1.3.0.zip` をダウンロード。
2.  解凍して `dist` フォルダを取り出す。
3.  Chromeで `chrome://extensions/` を開き、「デベロッパーモード」をON。
4.  「パッケージ化されていない拡張機能を読み込む」から `dist` フォルダを選択。
